i.matdoes.dev API

  • POST /api/upload

    Send data as {"image": <file data>}

    Recieves Json with the following attributes:
    hash - The hash of the image,
    url - URL for viewing the image,
    view - URL for viewing the image without html,
    password - Password used for deleting the image,
    delete - URL shortcut for deleting the image

  • GET /api/shorten/<hash>

    Returns the short code for the image in plain text

  • GET /api/delete/<password>

    Deletes the image if the password is correct
